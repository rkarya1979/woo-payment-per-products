Woocommerce Payment Per Products

== Description ==

Woocommerce Payment Per Products is the perfect solution for any WordPress website which needs payment method based on product. This plugin only works with woocommerce.


= Website =
http://www.indianbusybees.com/


== Installation ==

1. Upload 'woo-pay-per-pro' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In product add/edit page choose payment method for the checkout.
